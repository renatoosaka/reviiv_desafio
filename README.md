### Reviiv Backend Challenge

#### Instalar dependências

```
npm install
```

#### Executar o projeto

```
npm run dev
```

#### Acessar documentação

```
http://localhost:5000/reference
```

#### Gerar build

```
npm run build
```

#### Rodar testes

```
npm run test
```

#### Executar parser

```
npm run parser
```
